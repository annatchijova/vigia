Weighted Fraction score

CRONOS trace chain

One trace per agent + gate

Hypotheses, evidence, and each agent's own vote — every entry hashes the previous one (SHA-256). Retroactive edits break the chain.

Qwen narrator — read-only

Narrates, never judges

Reads the sealed verdict, writes the human-readable case + an auto-generated devil's-advocate counter-hypothesis. Cannot change a single bit of the verdict above.

Output

Narrated explanation, delivered alongside the sealed verdict and its tamper-evident trail — the explanation can be regenerated with a different model at any time without touching either.

One codebase, one pipeline. CORVUS (detection), the corroboration gate, CRONOS (audit chain), and the Qwen narration step are modules that run in a single process against a single text artifact — not separate services or repositories. The only external call in the whole flow is the Qwen narration step at the very end, and it happens after the verdict is already sealed.
